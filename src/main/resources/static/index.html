API Documentation

Explore the Git Integration API using our interactive API documentation. The documentation provides detailed information about available endpoints, request parameters, and response formats.

Open Swagger UI View OpenAPI Specification

  • GET /api/github/branches

    Fetches all branches from a GitHub repository

    Parameters: owner, repo

  • GET /api/github/commits

    Fetches commits from a GitHub repository

    Parameters: owner, repo, branch (optional), limit (optional)

  • GET /api/github/pull-requests

    Fetches pull requests from a GitHub repository

    Parameters: owner, repo, state (optional)

  • GET /api/gitlab/branches

    Fetches all branches from a GitLab repository

    Parameters: projectId

  • GET /api/gitlab/commits

    Fetches commits from a GitLab repository

    Parameters: projectId, branch (optional), limit (optional)

  • GET /api/gitlab/merge-requests

    Fetches merge requests from a GitLab repository

    Parameters: projectId, state (optional)

  • GET /api/git/branches

    Fetches all branches using either GitHub or GitLab integration

    Parameters: provider, repository

  • GET /api/git/commits

    Fetches commits using either GitHub or GitLab integration

    Parameters: provider, repository, branch (optional), limit (optional)

  • GET /api/git/pull-requests

    Fetches pull requests (GitHub) or merge requests (GitLab)

    Parameters: provider, repository, state (optional)
